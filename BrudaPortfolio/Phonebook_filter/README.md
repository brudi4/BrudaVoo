# Проект "Телефонная книга"
Программа получает на вход телефонную книгу с номерами, записанными в разных форматах, а на выходе выдает тот же список номеров, но в международном формате.

## Пример

Вход:
88965584713
81124792236
8(800)4548787
88005555555
+7-988-533-0029

Выход:
+78965584713
+71124792236
+78004548787
+78005555555
+79885330029
